Air Quality Prediction 
It predicts NO₂ (Nitrogen Dioxide) air quality levels based on real-world sensor data, built with Linear Regression and deployed using Streamlit.
This project uses the Air Quality Dataset to train a regression model that estimates air pollution levels from multi-sensor readings. The app takes sensor inputs from the user and returns a predicted air quality value along with a risk level classification (Good / Moderate / Unhealthy / Hazardous).
